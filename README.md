#关于本书
三维图形硬件正在迅速发展,已经不仅仅是计算机系统的一个主要议题,而是一个不可或缺的组成部分。许多操作系统直接使用,甚至必须要求某种程度的3D渲染硬件。即使在日益重要的移动计算领域,3D图形硬件已经成为一个标准特性甚至在那些配置非常低的设备上。

了解如何充分利用硬件是一个困难的挑战,特别是对于一个新的图形学和图形渲染。
